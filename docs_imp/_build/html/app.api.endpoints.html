app.api.endpoints.kpi module

async app.api.endpoints.kpi.get_kpi(request: KPIRequest) → KPIResponse[source]

Computes a KPI based on the provided request.

This endpoint performs the computation of a KPI using the provided parameters in the KPIRequest model. It will call the compute function from the dynamic engine to process the request and return the result as a KPIResponse. If there is an error during the computation, an appropriate HTTP exception is raised.

Parameters:

request (KPIRequest) – The input parameters required to compute the KPI.

Returns:

The computed KPI result, encapsulated in a KPIResponse.

Return type:

KPIResponse

Raises:

HTTPException –

  • If the computation fails due to invalid input (ValueError), raises a 404 status with the error message.

  • If an unexpected error occurs, raises a 500 status with a generic “Internal Server Error” message.

app.api.endpoints.real_time module

async app.api.endpoints.real_time.real_time_session(request: RealTimeKPIRequest) → RealTimeResponse[source]

Starts a real-time KPI session.

This endpoint initializes a real-time KPI session by setting up the Kafka consumer and KPI engine. It processes the provided request, validates inputs, prepares the KPI formula, and begins streaming data for real-time KPI computation.

Parameters:

request (RealTimeKPIRequest) – The request containing details of the KPI session, such as KPI name, machines, and operations.

Returns:

A response indicating the success or failure of starting the real-time session.

Return type:

RealTimeResponse

async app.api.endpoints.real_time.shutdown_event()[source]

Handles application shutdown by stopping the Kafka consumer and cleaning up resources.

This function is triggered during application shutdown. It stops the Kafka consumer and deletes the Kafka topic associated with the real-time session.

Raises:

Exception – If errors occur during the cleanup process.

async app.api.endpoints.real_time.stop_consumer() → RealTimeResponse[source]

Stops the currently running Kafka consumer.

This endpoint stops the real-time KPI session by signaling the Kafka consumer to terminate, processing any remaining data, and closing all active connections.

Returns:

A response indicating the success or failure of stopping the consumer.

Return type:

RealTimeResponse
